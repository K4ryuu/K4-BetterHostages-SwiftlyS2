## Installation

1. Install [SwiftlyS2](https://github.com/swiftly-solution/swiftlys2) on your server
2. [Download the latest release](https://github.com/K4ryuu/K4-BetterHostages-SwiftlyS2/releases/latest)
3. Extract to your server's `swiftlys2/plugins/` directory

<!-- FEATURES -->

## Features

- Both Terrorists and Counter-Terrorists can pick up hostages
- Players can drop hostages to tactically reposition them
- Timer-based pickup and drop system with progress tracking
- Sound feedback for pickup and drop actions
- Configurable pickup range and action durations
